# VS Code Extension

## Dev

```bash
cd apps/vscode-extension
npm install
npm run build
```

## Run / Debug

- If F5 changes brightness on your laptop, use `Fn+F5` or start debug from `Run and Debug` panel.
- Alternative: `Ctrl+Shift+D` then click `Start Debugging`.

## Settings

- `codebaseAgent.apiBase` (default: http://localhost:8000)
- `codebaseAgent.projectId` (repo id like r_123)

## Commands

- Codebase Agent: Explain Selection (`Ctrl+Shift+A`)
- Codebase Agent: Refactor Function (`Ctrl+Shift+R`)
